Welcome to my new blog, **Rational Writings**!

I wanted a blog that looked exactly like the environment I spend most of my time in: the terminal.

No flashy CSS frameworks, no complex build steps. Just pure, simple HTML and some raw, dark-mode CSS.

Here is a quick look at the font differences. This paragraph is rendered in a standard system monospace font, which is great for general reading while still retaining that typewriter feel.

```html
<!-- However, code blocks use a dedicated developer font -->
<!-- We specifically fall back to 'Fira Code', 'Cascadia Code', etc. -->
<div class="test">
  <p>Hello World</p>
</div>
```

Enjoy the simplicity!
